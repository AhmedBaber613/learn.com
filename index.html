Welcome!

Hello! I’m Cheshire, and this is my easy HTML/CSS tutorial. The purpose of this tutorial is to get you to a good starting point where you can have fun building your own websites with HTML and CSS. This
tutorial is geared towards non-programmers, so instead of going super in-depth and overwhelming you with technical details, it will tell you what you need to know to start building your own basic webpages fast.

To get the most out of this tutorial, make sure you follow along with the sections that say “try it yourself!” The best way to learn programming is by doing it.

Click on “Intro to HTML” in the sidebar to get started. Have fun!

Contact
